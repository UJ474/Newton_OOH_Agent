# GOLDEN RULES

1.

Never overwrite source data.

Always enrich.

---

2.

Every enrichment should be reusable.

---

3.

LLMs generate knowledge.

Code executes business logic.

---

4.

One responsibility per module.

---

5.

Search and ingestion are separate systems.

---

6.

Never embed raw database rows.

Embed semantic documents.

---

7.

Never trust LLM output.

Validate everything.

---

8.

Cache expensive operations.

---

9.

Everything should be explainable.

---

10.

Build an AI Media Planner,
not an AI Search Engine.

Search is only one capability.
Planning is the product.
